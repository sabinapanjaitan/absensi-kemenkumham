# SIAP (Sistem Informasi Absensi & Kegiatan)

Aplikasi absensi berbasis web untuk manajemen kehadiran dan kegiatan harian.

## Cara Instalasi
1. Pastikan **XAMPP** atau server lokal lainnya sudah terinstal dan berjalan (Apache & MySQL).
2. Buat database baru di `phpMyAdmin` dengan nama `db_absensi`.
3. Import file `db_absensi_structure.sql` ke dalam database tersebut.
4. Sesuaikan konfigurasi database di file `koneksi.php` dengan pengaturan lokalmu.
5. Jalankan melalui browser dengan mengakses alamat folder proyek ini.

## Teknologi yang Digunakan
- PHP Native
- MySQL
- CSS (Custom Design)
